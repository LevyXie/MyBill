# MyBill
JavaSE练手项目：一本糊涂账

项目来源：https://how2j.cn/module/104.html

## 主要知识点：
-  **编程基础：**
   面向对象，异常 ，集合，JDBC, 反射机制，I/O，Swing， 利用TableModel更新数据, 图形界面的皮肤

-  **软件设计思想：**
   单例模式，面板类与监听器类松耦合，Entity层设计，DAO层设计，Service层设计

- **业务常见处理手法：**
  CRUD操作，配置信息，配置信息初始化，报表生成，一对多关系，多对一关系
